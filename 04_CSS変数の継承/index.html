複数の要素で使いたい場合

CSS 変数は継承されるプロパティのため共通の親要素で指定する必要がある。
:rootはよく使われる記法。

注意点①

CSS 変数はプロパティの値に使える。プロパティ名のほうには使うことはできない。
たとえば :root で --my-property: color; と宣言したあとに、
h1 の color を var(--my-property) で置き換えようとしても、うまくいかない。

注意点②

CSS 変数では、あとから単位を追加することはできない。 たとえば、 :root で --my-margin-left: 32; と宣言したあとに、
h1 でmargin-left に対して、 var() 関数を使って --my-margin-left を入れた場合、単位がつけられずうまく動かない。
単位を指定したい場合は
①宣言時につける
②CSS の calc() 関数を使う
